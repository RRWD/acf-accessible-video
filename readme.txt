== Description ==

Add accessible video's in your content, using Advanced Custom Fields Pro and the native WordPress media functionality.


== Installation ==

= Manual installation =

Download the plugin zip file to your computer and unzip it
Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation's wp-content/plugins/ directory.
Activate the plugin from the Plugins menu within the WordPress admin.
